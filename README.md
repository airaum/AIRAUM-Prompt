# AIRAUM Prompt Refiner

한국어·영어의 거친 사용자 입력을 LLM이 더 잘 이해할 수 있는 구조적 프롬프트로 정제하는 웹 도구입니다.

서버 없이 브라우저만으로 동작하며, 결과는 바로 복붙할 수 있는 프롬프트 1개만 출력합니다.  
누구나 링크를 클릭하면 바로 사용할 수 있습니다. 설치, 로그인, 결제 없음.

## 이런 경우에 잘 맞습니다

- 질문이 막연해서 AI 답변이 흐릿할 때
- "잘 써줘"처럼 짧고 애매하게 입력했는데, 더 나은 결과를 받고 싶을 때
- 프롬프트를 직접 구조화하기 번거로울 때
- 초중급 사용자가 무료 LLM에서 더 좋은 답변을 얻고 싶을 때

## 사용 예시

| 입력 | 설명 |
|------|------|
| `마케팅 보고서 써줘` | 보고서 구조로 정제 |
| `시장조사해줘` | 조사 목적에 맞게 보정 |
| `인스타 업로드 컨셉 정해야해 아이디어 제시해줘` | 콘텐츠 기획 + 아이디어로 분류 |
| `파이썬 쉽게 설명해줘` | 설명 유형으로 정제 |

## 핵심 기능

- **산출물 유형 우선 라우팅** — 보고서, 기획안, 비교, 설명 등 산출물 유형을 먼저 파악하고, 그에 맞는 출력 구조를 적용합니다.
- **작업 유형 분류** — 조사, 작성, 발상, 학습 등 8가지 작업 범주를 감지하여 역할과 원칙을 결정합니다.
- **자동 보정** — 짧거나 모호한 입력은 목적, 형식, 기준 등을 보수적으로 보충합니다. 지역, 플랫폼, 예산 등 사용자 고유 정보는 추정하지 않습니다.
- **한국어/영어 자동 감지** — 입력 언어에 맞춰 프롬프트를 생성합니다.
- **다크 모드** — 시스템 설정을 따르거나 수동 전환할 수 있습니다.

## 안전 정책

- 이 도구는 안전하고 합법적인 범위의 요청만 정제합니다.
- 불법행위, 타인에게 피해를 주는 행위, 침해·착취·우회·사기·해킹 관련 요청은 처리하지 않습니다.
- 법률, 의료, 금융 등 민감 주제는 일반 정보 정리 수준으로만 제한적으로 처리되며, 확정적 자문을 대체하지 않습니다.

## 제한사항

- 한국어/영어 텍스트만 지원합니다.
- 파일 첨부, 이미지, 외부 검색은 지원하지 않습니다.
- 결과는 최종 프롬프트 1개만 출력합니다.
- 답변 생성기가 아닙니다. 프롬프트를 정제할 뿐, AI 답변 자체를 만들지 않습니다.
